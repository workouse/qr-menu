---
title: Kategori Ekleme
description: Menünüze nasıl kategori eklersiniz
---

Kategoriler ürünlerinizi gruplandırır (örn. "Başlangıçlar", "Ana Yemekler", "Tatlılar").

**Kategori eklemek için:**
1. Düzenlemek istediğiniz menüyü açın.
2. **Kategori Ekle** (Add Category) butonuna tıklayın.
3. Kategori adını ve isteğe bağlı bir açıklamayı girin.
4. **Kaydet**'e tıklayın.
5. Menü görünümünde kategorileri sürükleyip bırakarak sıralayabilirsiniz.
