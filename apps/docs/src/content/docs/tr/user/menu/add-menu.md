---
title: Menü Ekleme
description: Mekanınıza nasıl menü eklersiniz
---

Menüler, müşterilerinize sunduklarınızı organize eder. Birden fazla menünüz olabilir (örn. "Kahvaltı", "Akşam Yemeği", "İçecekler").

**Menü eklemek için:**
1. **Menüler** bölümüne gidin veya Mekan panelinizi açın.
2. **Menü Oluştur** (Create Menu) butonuna tıklayın.
3. Menünüz için bir başlık girin (örn. "Ana Menü").
4. Hemen aktif olup olmayacağını seçin.
5. **Kaydet**'e tıklayın. Artık menünüze kategori ve ürünler eklemeye hazırsınız!
