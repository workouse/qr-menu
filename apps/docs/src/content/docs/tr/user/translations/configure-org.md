---
title: Çoklu Dil Ayarları
description: Organizasyonunuz için dilleri ayarlama
---

QR Menü, müşterilerinize birden fazla dilde hizmet vermenizi sağlar.

**Çoklu dil desteğini etkinleştirmek için:**
1. Ana panelden **Ayarlar** (Settings) bölümüne gidin.
2. **Diller** (Languages) bölümünü bulun.
3. Organizasyonunuz için varsayılan dili seçin.
4. Desteklenen ek dilleri ekleyin (örn. İngilizce, Almanca, İspanyolca).
5. **Kaydet**'e tıklayın.

Artık menüleriniz boyunca bu diller için çeviriler ekleyebileceksiniz!
