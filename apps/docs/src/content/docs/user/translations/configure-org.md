---
title: Configure Multilanguage
description: Setup languages for your organization
---

QR Menu allows you to serve customers in multiple languages.

**To enable multilanguage support:**
1. Go to your **Settings** from the main dashboard.
2. Look for the **Languages** section.
3. Select the default language for your organization.
4. Add any additional supported languages (e.g., Turkish, German, Spanish).
5. Click **Save**. 

Now you will be able to add translations for these languages across your menus!
