---
title: Add Translations
description: Translate menus, categories, and items
---

Once you've configured multiple languages for your organization, you can translate your content.

**To add a translation:**
1. Open any Menu, Category, or Item you want to translate.
2. Look for the **Translations** tab or the language switcher dropdown.
3. Select the language you want to translate into (e.g., "Türkçe").
4. Enter the translated text for the Name and Description fields.
5. Click **Save**.

When a customer scans your QR code, they can choose their preferred language, and the system will automatically display the translations you provided.
