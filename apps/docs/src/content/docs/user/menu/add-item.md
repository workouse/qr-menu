---
title: Add an Item
description: How to add items to a category
---

Items are the individual products your customers will order (e.g., "Cheeseburger", "Cola").

**To add an item:**
1. Open your menu and locate the category where you want to add the item.
2. Click **Add Item** under that category.
3. Fill in the details:
   - **Name**: e.g., "Classic Cheeseburger"
   - **Description**: What's in it?
   - **Price**: Set the item price.
   - **Image**: Upload an appetizing photo!
4. Click **Save**. Your item is now added to the category.
