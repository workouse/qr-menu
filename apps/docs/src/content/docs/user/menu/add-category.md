---
title: Add a Category
description: How to add a category to your menu
---

Categories group your items together (e.g., "Starters", "Main Courses", "Desserts").

**To add a category:**
1. Open the menu you want to edit.
2. Click the **Add Category** button.
3. Enter the category name and an optional description.
4. Click **Save**.
5. You can drag and drop categories to reorder them in the menu view.
